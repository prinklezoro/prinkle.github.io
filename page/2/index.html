使用template元素定制客户端模板

  • 检测浏览器是否支持template元素
  • template元素中的内容不会被浏览器渲染，必须调用才开始渲染
  • 调用之前，模板中的内容并不看作为页面DOM树的内容。使用document.getElement等方法获取不到模板中的元素
  • 模板不可能被预渲染
  • 模板可以放置在任何位置
  • template.content是一个只读属性，包含了模板的DocumentFragment对象
  • 可以在模板中嵌套模板，但是激活外层模板并不会激活内层模板

通信API

发表于 2019-09-01 分类于 编程 ， 前端 Valine：

跨文档消息传输

  • HTML5提供了网页文档之间互相接收和发送消息的功能。使用这个功能，只要获取到网页所在窗口的对象，不仅同源，跨域也能实现通信
  • 接收从其他窗口传来的消息：window.addEventListener(‘message’, fn, false)
  • 向其他窗口发送消息：otherWindow.postMessage(message, targetUrl, [ports])。otherWindow为要发送窗口的window对象的引用，可以通过window.open返回该对象，也可以通过window.frames[index]。第一个参数为消息文本（字符串）；第二个参数为要发送的url地址 *表示所有地址；第三个参数表示传入的端口号
  • window.frames数组获取页面中所有iframe标签所在的window对象的集合
  • 也可以使用iframeDom.contentWindow来获取响应的window对象

Web Storage

Web Storage的概念

  • cookie的缺点
    • cookie的大小被限制在4K
    • cookie是岁HTTP事务一起被发送的。因此会浪费一部分发送cookie时使用的带宽
    • 操作cookie很复杂
  • sessionStorage：将数据保存到session中。session的生命周期是指用户从进入网站到关闭浏览器这一段时间。
  • localStorage：将数据保存到客户端本地的硬件设备，浏览器关闭了还是会存在
