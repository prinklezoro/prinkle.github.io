ES6新增功能

发表于 2019-08-20 分类于 编程 ， 前端 Valine：

这里只是简介 详细笔记请参考ES6笔记

新增语法

for of循环

for of循环可以对任何集合（具备iterator属性）进行循环

let关键字和const关键字

var声明的变量会提前，let不会；var声明的变量没有块级作用域（if里面的内容也属与；但是有函数作用域），let有块级作用域

for循环中使用let变量，为每一次循环都创建一个变量，而不是var声明的只有一个变量。这样就每次循环，循环体都会产生一个x的副本。for循环括号的作用域是循环体作用域的父级

const声明变量的同时必须赋值，并且不允许二次赋值

新增类

类的使用方法

类的静态方法

继承

不确定参数

函数默认参数

  • 默认参数可以是一个函数、表达式或者三元运算符
  • 传入undefined等于没有传入值（使用默认值）
  • 不被指定默认参数值的参数值为undefined

箭头函数

  • 当函数体只有一行时，可以省略{}和return
  • 当需要返回一个对象时，可以用括号包住（不包住会当成一个函数作用域的大括号）
  • this继承父函数，箭头函数没有this对象

generator函数

  • 以function*开始，调用一个生成器函数时，函数不被立即执行，只是返回一个被暂停的generator对象（具备迭代器属性）
  • generator对象有一个next方法，每次调用，函数体代码继续执行，直到下一个yield语句
  • next()方法的返回值是一个对象，具备done和value属性，done判断对集合的访问是否结束，value值是yield后面的返回值
  • yield表达式并没有值（undefined），但是在下一次调用next方法的时候可以为传入参数，代表上一个yield表达式的
  • generator对象可以使用for of循环

解构赋值

  • 数组的解构赋值
  • spread运算符
  • 在解构赋值中运用spread运算符
  • 对象的解构赋值

变量名必须跟属性名相同

  • 数组对象嵌套的解构赋值

注意，对象解构赋值再次赋值要用括号括起来

模板字符串——标签化模板

客户端js模块化

  • 使用export或者export defalut向外暴露变量和函数
  • 使用import from导入模块

export 和 export defalut的区别

  1. export可以导入很多次，export defalut只能导入一次
  2. 接受的时候属性名必须相同，export defalut只能用一个变量接受，名字随意

注意点

  1. 只能在顶级作用域使用import关键字以及export关键字。不允许在条件判断或者循环或者函数中使用
  2. 所有导出必须指定导出名。不允许在循环中多次导出
  3. 模块对象是被冻结的，不允许在模块外部对模块添加新的特性
  4. 不能对模块进行try catch处理
  5. 在模块的依赖被加载前，不允许运行模块的任何代码。模块不控制如何加载其中的依赖

新增对象及数据类型

promise对象

symbol对象

symbol类似于字符串，作为一个唯一的标识符而存在

  • 创建 symbol 对象
  • Symbol()方法可以传参，只是为了在控制台输出便于区分
  • Symbol.for(‘’)，首先在symbol注册表中搜索有没有以该参数为名称的symbol，如果有，就返回这个symbol，否则就创建并返回一个以改字符串为名字的symbol对象

Symbol.for(‘’)与Symbol(‘’)不同，后者只是为了输出便于区分
Symbol.keyfor(symbol变量) 获取登记的symbol名称

  • Symbol.iterator：一个数据类型要使用for of循环或者spread运算符 则必须有Symbol.iterator属性，并为其定义方法
  • 不能用.符号来访问以symbol为属性名的属性，必须使用方括号来访问

代理

  • 代理对象内有两个参数，第一个是目标对象，第二个是包含内置函数的对象。当代理对象执行某些操作（赋值，取值），会触发相应的内置函数
  • proxy对象内置函数
名称 调用时机
get(target,key,receive) 第一个参数代表目标对象；第二个代表对象属性；第一个代表proxy实例。获取属性值被调用
set(target, key, value, receive) 第二个参数代表对象属性；第三个代表需要设置的值。设置属性值被调用
has(target, key) 判断对象是否具有某个属性，在key in obj 中被内部调用
getPrototypeOf(target) 用于获取对象的原型,在obj.proto或者getPrototypeOf()中调用
setPrototypeOf(target, prototype) 用于设置对象的原型，在setPropertyOf(prototype)中调用
isExtensible(target) 判断是否可对对象添加属性 Object.isExtensible(obj)被内部调用
preventExtensible(target) 阻止对对象添加属性，Object.preventExtensiable(obj)调用
getOwnPropertyDescriptors(target, key) 获取一个对象属性的属性描述 Object.getOwnPropertyDescriptors(obj, key)被内部调用
defineProperty(target, key, propertyDescriptor) 创建一个对象的属性或者修改属性是被调用，Object.defineProperty(obj,key,pd)
delete(target, prop) 删除对象的属性被调用，delete obj.prop
ownKeys(target) 用于获取对象的所有属性名，Object.getOwnPropertyNames(obj)或者Object.getOwnPropertySymbols(obj)被内部调用
apply(target, ctx, arguments) 第一个参数是目标对象（函数）；第二个参数是函数被调用的对象。当函数被调用时触发
construct(target, arguments, newTarget) new 对象时触发
  1. construct(target,args,newTarget) 第三个参数返回的是一个函数里面包含proxy对象的两个参数

native code 表示无法正常调用，只有通过抛出的接口来调用 待续…

Set集合

为什么要增加集合的概念

  1. 开发者必须小心将对象的方法转换为对象的键名
  2. 属性的键名不能为对象
  3. 不能得知一个对象具有多少属性
  4. 不能遍历对象
  • set集合不会保存重复的元素
  • add() 添加元素
  • clear() 移动所有元素
  • delete(value) 移动对象中与这个值相等的元素 返回true或者false
  • entries() 返回一个新的迭代器对象 包含键名和键值
  • keys() 返回一个新的迭代器对象 包含键名
  • values() 返回一个新的迭代器对象 包含键值
  • forEach(callback([,thisArg])) 遍历。第一个参数是值，第二个参数是回调函数的this对象
  • has(value) 返回一个布尔值，表示指定值是否存在 返回true或者false

set元素的键名和键值是一样的
NaN只能添加一次

  • Array.from(set)方法将set集合转换为数组 new Set()里面的参数也可以是一个数组，也可以是一个字符串

Map集合

  • 由一些键名/键值对组成的集合。键名可以是非字符串和Symbol对象
  • get(key) 根据给定键名返回键值 不存在返回undefined
  • set(key, value) 添加一个键名为key 值为value的属性。如果存在则修改
  • clear() 移动所有的键值对
  • has(key) 是否包含此键名
  • delete(key) 删除键名是key的属性
  • entries() 返回一个新的迭代器对象，包含Map对象中的所有键值对
  • keys() 返回一个迭代器对象，包含Map对象中的所有键名
  • values() 返回一个迭代器对象，包含Map对象中的所有键值
  • forEach(value, key, this) 遍历 this代表map对象
  • 使用二维数组创建map对象

WeakSet和WeakMap

  • 两个都是弱引用，即当键值对中的元素被删除，WeakSet和WeakMap中对应的键值对也被删除
  • 只能对WeakSet集合使用new、has、add和delete方法（不能遍历）
  • 只能对WeakMap集合使用new、has、get、set以及delete方法（不能遍历）
  • 存储在WeakSet集合中的数据必须为对象，存储在WeakMap集合的键名必须为对象

对现有对象的扩展

Number对象

  • isInteger 判断是否为整数
  • isNaN 判断是否为NaN

String对象

  • padStart(targetLength, padString) 第一个参数是目标的长度，第二个参数是用于填充的字符串，默认为空格
  • padEnd(targenLength, padString) 同上

Array对象

  • Array.from(arrayLike, mapFn, thisArg) 将一个伪数组或者可迭代的对象转换成一个新的数组。第一个参数是要转化的伪数组或其他可迭代的对象；第二个参数是一个回调函数（有2个参数，value key）对数组中的每一个元素调用该回调函数；第三个参数是调用第二个参数的this对象
  • Array.of(…args) 使用不确定的参数创建新数组并返回
  • fill(value, start, end) 填充一个数组从起始索引到终止索引的全部元素，并将原数组返回。第一个参数：用来填充的值；第二个参数：用于指定填充的起始索引，默认为0，如果是负数，数组长度+起始索引开始；第三个参数：用于指定填充的终止索引（不包括），默认是数组长度，如果是负数，计算为数组长度+参数值
  • find(callback)和findIndex(callback) 返回该元素和返回该元素索引
  • copyWithin(target, start, end) 复制数组的一部分到另一个位置。第一个参数表示目标起始索引位置，如果是负数计算为数组长度+参数值；第二个参数：复制源的起始索引 默认为0；第三个参数：复制源的终止索引，默认是数组长度
  • entries和keys方法

Object对象

  • assign(target, …source) 合并对象到第一个参数对象中，并返回第一个参数对象

后面的对象如果键名相同会覆盖前面的
