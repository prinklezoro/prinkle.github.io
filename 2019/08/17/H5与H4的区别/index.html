H5与H4的区别

发表于 2019-08-17 更新于 2019-08-19 分类于 编程 ， 前端 Valine：

H5与H4的区别

H5确保的兼容性

可以省略标记的元素

  • 不允许写结束标记的元素有：area br hr img input link meta param
  • 可以省略结束标记的元素有：li dt dd p thead tbody tr td th
  • 可以省略全部标记的元素有：html head body colgroup tbody

具有boolean值的属性

例如disabled readonly等，当只写属性名，表示true。也可以使用属性名或者空字符串作为属性值

省略引号

指定属性值的时候既可以用双引号，也可以用单引号，当属性值不包括空字符串，> < = 单引号和双引号时，引号也可以省略

全局属性

contentEditable

contentEditable为true时，元素可以编辑；为false，元素不可以编辑

designMode

designMode为on 页面中的元素可以编辑；designMode为off 页面中的元素不可以编辑

只能用在script脚本中，作为document的属性使用，document.designMode = on

tabindex属性

不断敲击tab键让窗口或页面中的元素获得焦点，每一个控件的tabindex表示该控件是第几个被访问到的

有些元素可以通过设置tabindex属性，从而可以利用focus()方法获取焦点，但是tab也能获取焦点，这时可以使用tabindex = -1

新增事件

window或者body对象:
beforeprint: 即将开始打印之前触发
afterprint: 打印完毕触发
resize: 浏览器窗口发生改变时触发
error: 页面加载出错时触发
offline 页面变为离线状态时触发
online 页面变为在线状态时触发
pageshow 页面加载时触发 load时间只在第一次加载触发，缓存中加载只触发pageshow
beforeunload 页面被关闭时触发
hashchange 页面#后面的值改变时触发
任何元素:
mousewheel 鼠标指针悬停在元素上并且滚动鼠标滚轮时触发
任何容器元素:
scroll 元素滚动条被滚动时触发
input元素或者textarea元素:
input: 用户修改文本框内容触发，change用户失去焦点才触发，input不需要失去焦点
input元素或者textarea元素:
reset 当用户按下reset按钮或者触发reset方法时触发

只触发一次事件

addEventListener: 第三个参数 {once: true}
