盒类型

block inline inline-block

如何去除行内块元素换行所带来的空白占位符

  • 将父容器的字体大小设置为0
  • 给子元素浮动效果
  • 注释块级元素周边的空白区域
  • 行内块元素一行显示，不换行
  • 推荐使用第一种方法

inline-table类型

  • 默认table标签时块级元素，当设置inline-table 变成行内元素。也就是变成内联样式和table样式
  • 当设置为inline-table时，默认对齐方式是居中对齐

list-item类型

  • 将元素的类型设定为列表类型来显示，相当于ul ol
  • 使用list-style将列表标记属性
    • list-style-type 设置标记的类型
    • list-style-position 属性值为inside和outside(默认)
    • list-style-image 使用图像来替换列表项的标记

  • list-style-type的属性值

table以及table-cell

table的特性以及相关属性的设置

  • 与浮动同级，浮动的元素并不会浮在table标签之上（具体原因找了很久没找出来…）

上图浮动的元素已经设置了透明度，可以看到底下并没有table元素

  • table-layout属性显示表格单元格、行、列的算法规则
    • fixed：固定表格布局。如果指定了单元格宽度，那么会按照指定的宽度限定没有单元格宽度，与内容无关；如果没有指定单元格宽度，会根据table的总宽度平分到每个单元格
    • auto（默认）：自动表格布局。如果指定了单元格宽度，那么当内容不够指定宽度时，宽度就是单元格宽度，如果单元格宽度超过了table宽度，则宽度就是table宽度（也就是说无论单元格宽度设置多大，始终不会超过table宽度，除非内容撑开了table宽度）；如果没有指定单元格宽度，那么表格宽度完全由内容决定

table宽度的研究

  • 似乎table的最大宽度就是剩余宽度，但远没有这么简单

  • 当table内容超过了剩余宽度时，table会另起一行，似乎最大宽度为父元素宽度；但是当我继续加内容，table宽度会无限增长，下面会出现滚动条
  • 总结：
    • 当table-layout默认时，单元格的内容可以决定table宽度。
    • 内容不足单元格宽度时，单元格宽度为指定宽度；
    • 内容超过，单元格宽度为内容撑开的宽度；
    • 内容不足剩余宽度，table不会另起一行；
    • 内容超过剩余宽度，table另起一行。相当于行内块元素，但是与浮动元素平级

table-cell

  • 两个特点：同行等高；宽度自动调节
  • css有一个规则，创建匿名表格元素。也就是说当某个元素被设置为table-cell时，当他的父元素，祖父元素没有table-row和table时，会跟随它相邻的table-cell元素一起生成父元素table-row和祖父元素table
  • 同行等高

因为display:table-cell外面会套一层table匿名元素，所有会形成滚动条

盒中容纳不下的内容的显示

overflow属性

  • overflow:auto; overflow:hidden; overflow:srcoll
  • overflow-x:hidden; overflow-x:srcoll
  • overflow-y:hidden; overflow-y:srcoll

text-overflow属性

  • 当内容超出盒子内容时使用该属性，通常与overflow一起用
  • text-overflow:ellipse，会在末尾显示省略符号
  • text-overflow:clip，修剪文本 默认属性
  • text-overflow:string 只有火狐浏览器支持

  • 一行显示省略号
  • 多行显示省略号

box-shadow

  • box-shadow: x-shadow y-shadow blur color inset
  • x-shadow：水平阴影的位置，允许负值，必须
  • y-shadow：垂直阴影的位置，允许负值，必须
  • blur：模糊距离。可选
  • color：可选，颜色 默认是黑色
  • inset 内阴影

CSS3盒模型

  • box-sizing 有两种属性值
  • box-sizing：content-box 标准模式下的盒模型，padding和border会改变width和height的值
  • box-sizing：border-box 怪异模式下的盒模型，定义padding和border不会改变width和height值
