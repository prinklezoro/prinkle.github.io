文字和字体相关样式

发表于 2019-09-10 分类于 编程 ， 前端 Valine：

text-shadow

  • text-shadow: length length length color 给字体添加文字阴影.前三个length表示阴影的横方向距离、阴影的纵方向距离、阴影的模糊半径、color阴影的颜色
  • 前两个参数可以指定负值
  • 在谷歌、火狐、欧朋浏览器中可以给text-shadow指定多个文字阴影

css3换行相关属性

浏览器默认的换行规则

  • 浏览器本身都具有让文本自动换行的功能。对于中文文字，浏览器可以在任何一个中文字符换行；对于西方文字，浏览器会在半角空格或连字符的地方换行，不会断开一个单词换行。
  • 当含有标点符号的时候，标点符号不会出现在行首
  • nbsp;和
    都可以正常发挥作用，连续的空格被缩减为1个，换行符无效
  • 长度超过一行的单词会超过边界

white-space

  • 控制空白符的显示 还能控制是否换行，五个值 normal/nowrap/pre/pre-wrap/pre-line
  • white-space: no-wrap 强制一行显示，只有
    能换行

  • white-space: pre 空格和换行符全部被保留，但是自动换行没了

  • white-space: pre-wrap 保留空格和换行符，且可以自动换行

  • white-space: pre-line 空格被合并，但是换行符可以发挥作用，自动换行也有

word-break

  • 控制单词如何被拆分换行的，三个值 normal | break-all | keep-all
  • keep-all：只有空格连字符-或任何标点符号可以触发换行，所有单词或连续的中文文字都不能拆分换行

  • break-all：所有单词碰到边界一律触发换行

word-wrap(overflow-wrap)

  • 控制单词如何被拆分换行的，word-break的补充，两个值 normal | break-word
  • break-word：只有当一个单词一整行都显示不下时，才会拆分换行该单词

总结

属性 描述 属性值
white-space 控制空白字符的显示 normal / nowrap / pre / pre-wrap / pre-line
word-break 控制单词如何被拆分换行 normal / break-all / keep-all
word-wrap（overflow-wrap） 控制长度超过一行的单词是否被拆分换行，是word-break的补充 normal / break-word

use-select属性

  • 用于指定用户是否可通过拖拽鼠标来选取元素文字。
  • 三个属性：none（禁止选取）；text（可以选取）；all：只能选取全部文字，点击就选取全部文字

服务器端字体——@font-face

使用@font-face

  • 可以利用@font-face来定义服务器端字体或者字体图标
  • font-family后面的属性值可以随意定义，往后在该项目中，这个引用字体的名字就是myFont
  • src url() 用来指定服务器端字体文件所在的路径；format() 用来指定该字体文件的类型
  • format：字体的格式
format属性 对应的字体文件格式
truetype 常见的字体格式，对应文件为.ttf
opentype 对应文件.otf
woff 对应文件 .woff truetype和opentype的压缩版
embedded-opentype IE专用格式

为了解决兼容性问题，可以使用多个url+format属性值，用逗号隔开

  • 使用服务器字体时，就不会再为样式定义font-style或font-weight了，否则会失效
  • 如果一定要使用font-style或font-weight，则必须在font-face中定义对应的样式

显示客户端本地字体

  • src: local(‘名字’)

使用字体图标

  • 推荐一个网站，可以使用网站上已有的字体图标，也可以自己上传svg图片生成字体图标 https://icomoon.io/

icomoon网站生成字体图标使用方法

  • 点击icomoon app

  • 选择你想要的字体图标，或者点击import icons 上传你电脑上的svg图标文件。选择好了，点击 generate font

  • 点击 download 即可下载对应的字体文件压缩包

  • 复制font文件夹和style.css样式文件到项目目录 style.css可以改名字

  • 引入style.css文件，在元素上面加上对应的类名就能应用对应的字体图标

font-size-adjust

  • 该属性只有火狐浏览器支持
  • font-size-adjust设置的是字体自带的aspect值，计算公式为x-height值除以该字体的font-size，x-height表示这个字体书写出来的小写字母x的高度：例如font-size为100px的x-height为50px，则aspect为0.5
  • 常见西方字体的参照表
字体种类 aspect 字体种类 aspect
Verdada 0.58 Times New Roman 0.46
Comic Sans MS 0.54 Gill Sans 0.46
Trebuchet MS 0.53 Bernhard Modern 0.4
Georgia 0.5 Caflisch Script Web 0.37
Myriad Web 0.48 Fjemish Script 0.28
Minion Web 0.47
  • 相同大小字体下，每种文字的高度都不一致，可能会使页面布局混乱，这样就需要调整aspect值来达到一致高度
  • 要想保持高度一致，则x-height必须一致，那么字体值1 * aspect1 = 字体值2 * aspect2 必须成立

rem定义字体大小
